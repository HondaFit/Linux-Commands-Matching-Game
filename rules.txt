------- Welcome to the Matching Linux Commands Game! -------
---- Here are some rules ----
Each level you will be given a linux command and a list of command descriptions 
Your goal is to trying math the linux command to the corresponding description
If you get it right, you will get a point
If you get it wrong, you will loose a point
Once you reach 30 points, you win! 

---- Game Menu ---- 
When you first start the game, you will be given an option to create an account or load your account
This means that you and other users can play this game!!

If you want to add new linux commands along with linux descriptions, then you could do so
You can also view all the commands along with the descriptions in the csv file
